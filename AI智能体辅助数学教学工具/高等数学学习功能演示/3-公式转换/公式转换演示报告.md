# 一元微积分公式转换演示报告

## 📋 项目概述

本项目生成了一元微积分中最重要的10个公式的多种格式转换演示，包括导数、积分、极限和级数等核心概念。

## 📊 涵盖内容

### 1. 导数公式 (4个)
- 导数定义：f'(x) = lim(h→0) [f(x+h) - f(x)] / h
- 幂函数导数：d/dx (x^n) = n * x^(n-1)
- 乘积法则：d/dx [u(x) * v(x)] = u'(x) * v(x) + u(x) * v'(x)
- 链式法则：d/dx f(g(x)) = f'(g(x)) * g'(x)

### 2. 积分公式 (3个)
- 牛顿-莱布尼茨公式：∫[a to b] f(x) dx = F(b) - F(a)
- 基本积分公式：∫ x^n dx = x^(n+1) / (n+1) + C (n ≠ -1)
- 分部积分公式：∫ u dv = u*v - ∫ v du

### 3. 极限公式 (2个)
- 极限定义：lim(x→a) f(x) = L
- 重要极限：lim(x→0) sin(x) / x = 1

### 4. 级数公式 (1个)
- 泰勒展开：f(x) = f(a) + f'(a)(x-a) + f''(a)/2! * (x-a)^2 + ... + f^(n)(a)/n! * (x-a)^n + R_n(x)

## 🔄 转换格式

每个公式都提供了以下四种格式：

### 1. 📝 LaTeX格式
- 标准的数学排版语言
- 适用于学术论文和科技文档
- 语法：``$$公式$$``` 或 ``$公式$``

### 2. 🌐 MathML格式
- W3C推荐的网页数学标记语言
- 支持网页直接渲染
- 语义化的数学表达式表示

### 3. 📄 文本格式
- 纯文本表示方法
- 适用于不支持数学渲染的环境
- 保持数学表达式的可读性

### 4. 🎨 可视化显示
- 使用MathJax渲染的美观数学公式
- 支持交互式查看
- 专业的数学排版效果
