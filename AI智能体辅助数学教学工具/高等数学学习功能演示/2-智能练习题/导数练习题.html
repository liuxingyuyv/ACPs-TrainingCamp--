题目：求函数 f(x) = 3x² + 2x - 1 的导数
答案：f'(x) = 6x + 2
解析：使用幂函数求导法则：(x^n)' = nx^(n-1)，所以 (3x²)' = 6x，(2x)' = 2，(-1)' = 0

题目：求函数 f(x) = x³ - 4x + 7 的导数
答案：f'(x) = 3x² - 4
解析：逐项求导：(x³)' = 3x²，(-4x)' = -4，(7)' = 0

题目 3 中等
题目：求函数 f(x) = (2x + 1)³ 的导数
答案：f'(x) = 6(2x + 1)²
解析：使用链式法则：设u = 2x + 1，则f(x) = u³，f'(x) = 3u² · u' = 3(2x + 1)² · 2 = 6(2x + 1)²

题目 4 中等
题目：求函数 f(x) = x² · sin(x) 的导数
答案：f'(x) = 2x · sin(x) + x² · cos(x)
解析：使用乘积法则：(uv)' = u'v + uv'，所以 f'(x) = (x²)' · sin(x) + x² · (sin(x))' = 2x · sin(x) + x² · cos(x)

题目 5 困难
题目：求函数 f(x) = (x² + 1) · e^(-x) 的导数
答案：f'(x) = 2x · e^(-x) - (x² + 1) · e^(-x)
解析：使用乘积法则和链式法则：f'(x) = (x² + 1)' · e^(-x) + (x² + 1) · (e^(-x))' = 2x · e^(-x) + (x² + 1) · (-e^(-x))
