题目 1 简单
题目：一个球从高度h米自由落下，其运动方程为h(t) = 20 - 5t²，求2秒时的速度
答案：-20 m/s
解析：速度是位移对时间的导数：v(t) = h'(t) = -10t，当t = 2时，v(2) = -20 m/s（负号表示向下运动）

题目 3 中等
题目：一个长方形的长以3cm/s增加，宽以2cm/s减少，求当长为10cm，宽为6cm时，面积的变化率
答案：-2 cm²/s
解析：面积A = lw，dA/dt = l·dw/dt + w·dl/dt = 10·(-2) + 6·3 = -20 + 18 = -2 cm²/s（面积在减少）

题目 4 中等
题目：某产品的成本函数为C(x) = 1000 + 50x + 0.1x²，求生产100个单位时的边际成本
答案：70
解析：边际成本是成本函数的导数：C'(x) = 50 + 0.2x，当x = 100时，C'(100) = 50 + 20 = 70

题目 5 困难
题目：一个15米长的梯子靠在墙上，梯子底部以2m/s的速度滑离墙壁，求当梯子底部距离墙壁5m时，梯子顶部下滑的速度
答案：-0.5√5 m/s
解析：设梯子底部距墙x米，顶部高y米，则x² + y² = 225。对时间求导：2x·dx/dt + 2y·dy/dt = 0，当x = 5时，y = √(225 - 25) = √200 = 10√2，代入得：dy/dt = -(x/y)·dx/dt = -(5/(10√2))·2 = -1/√2 = -√2/2 m/s
