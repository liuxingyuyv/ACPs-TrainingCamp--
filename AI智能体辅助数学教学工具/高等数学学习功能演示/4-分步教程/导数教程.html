导数教程 - 从定义到应用

第一章：导数的定义

1.1 导数的几何意义

导数表示函数在某一点处的瞬时变化率，几何上表现为函数图像在该点处切线的斜率。

f'(x₀) = limh→0 [f(x₀ + h) - f(x₀)] / h

这个极限如果存在，就称函数 f(x) 在点 x₀ 处可导，其极限值就是导数。

1.2 导数的物理意义

在物理学中，导数具有重要的物理意义：

  • 位移对时间的导数 = 速度
  • 速度对时间的导数 = 加速度
  • 路程对时间的导数 = 瞬时速率

第二章：基本求导法则

2.1 基本初等函数的导数

(xn)' = nxn-1
(sin x)' = cos x
(cos x)' = -sin x
(ex)' = ex
(ln x)' = 1/x

2.2 求导的四则运算法则

(u ± v)' = u' ± v'
(uv)' = u'v + uv'
(u/v)' = (u'v - uv')/v²

2.3 复合函数求导法则

[f(g(x))]' = f'(g(x)) · g'(x)

这就是著名的链式法则，是求复合函数导数的关键。

第三章：典型例题演示

例题1：多项式函数求导

题目：求函数 f(x) = 3x³ - 2x² + 5x - 1 的导数

解答：
f'(x) = (3x³)' - (2x²)' + (5x)' - (1)'
f'(x) = 3 · 3x² - 2 · 2x + 5 · 1 - 0
f'(x) = 9x² - 4x + 5

例题2：三角函数求导

题目：求函数 f(x) = x²sin(x) 的导数

解答：
使用乘积法则：(uv)' = u'v + uv'
设 u = x²，v = sin(x)
u' = 2x，v' = cos(x)
f'(x) = u'v + uv' = 2x · sin(x) + x² · cos(x)
f'(x) = 2x sin(x) + x² cos(x)

例题3：复合函数求导

题目：求函数 f(x) = sin(3x² + 2x - 1) 的导数

解答：
使用链式法则：[f(g(x))]' = f'(g(x)) · g'(x)
设 f(u) = sin(u)，g(x) = 3x² + 2x - 1
f'(u) = cos(u)，g'(x) = 6x + 2
f'(x) = cos(3x² + 2x - 1) · (6x + 2)
f'(x) = (6x + 2)cos(3x² + 2x - 1)

第四章：导数的应用

4.1 函数的单调性

通过导数的符号可以判断函数的单调性：

  • 若 f'(x) > 0，则函数 f(x) 在该区间上单调递增
  • 若 f'(x) < 0，则函数 f(x) 在该区间上单调递减
  • 若 f'(x) = 0，则函数 f(x) 在该点可能有极值

4.2 函数的极值

极值的必要条件：若函数 f(x) 在点 x₀ 处可导且取得极值，则 f'(x₀) = 0

极值的充分条件：

  • 若 f'(x₀) = 0 且 f''(x₀) > 0，则 x₀ 为极小值点
  • 若 f'(x₀) = 0 且 f''(x₀) < 0，则 x₀ 为极大值点

例题4：求函数的极值

题目：求函数 f(x) = x³ - 3x² + 2 的极值

解答：
第一步：求导数 f'(x) = 3x² - 6x
第二步：令 f'(x) = 0，得 3x² - 6x = 0
3x(x - 2) = 0，所以 x = 0 或 x = 2
第三步：求二阶导数 f''(x) = 6x - 6
第四步：判断极值
• 当 x = 0 时，f''(0) = -6 < 0，所以 x = 0 为极大值点
• 当 x = 2 时，f''(2) = 6 > 0，所以 x = 2 为极小值点
极大值：f(0) = 2
极小值：f(2) = 8 - 12 + 2 = -2

第五章：练习题

练习1：基础求导

求下列函数的导数：

  1. f(x) = 4x⁴ - 3x³ + 2x² - x + 5
  2. f(x) = x²ex
  3. f(x) = ln(x² + 1)
  4. f(x) = cos(2x + 1)
答案：
1. f'(x) = 16x³ - 9x² + 4x - 1
2. f'(x) = 2xex + x²ex = (2x + x²)ex
3. f'(x) = 2x/(x² + 1)
4. f'(x) = -2sin(2x + 1)

练习2：极值问题

求函数 f(x) = x³ - 6x² + 9x + 1 的极值点和极值。

答案：
f'(x) = 3x² - 12x + 9
令 f'(x) = 0：3x² - 12x + 9 = 0
x² - 4x + 3 = 0
(x - 1)(x - 3) = 0，所以 x = 1 或 x = 3
f''(x) = 6x - 12
f''(1) = -6 < 0，所以 x = 1 为极大值点，极大值 f(1) = 1 - 6 + 9 + 1 = 5
f''(3) = 6 > 0，所以 x = 3 为极小值点，极小值 f(3) = 27 - 54 + 27 + 1 = 1

练习3：应用题

一个球的半径以 2cm/s 的速度增加，当半径为 10cm 时，球的体积增加速度是多少？

答案：
球的体积公式：V = (4/3)πr³
对时间t求导：dV/dt = 4πr² · dr/dt
已知 dr/dt = 2cm/s，r = 10cm
dV/dt = 4π(10)² · 2 = 4π × 100 × 2 = 800π cm³/s
所以当半径为10cm时，体积增加速度为800π cm³/s
