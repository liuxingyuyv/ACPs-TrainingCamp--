积分教程 - 不定积分和定积分

第一章：不定积分

1.1 原函数与不定积分的概念

如果函数 F(x) 的导数等于 f(x)，即 F'(x) = f(x)，那么 F(x) 称为 f(x) 的原函数。

∫f(x)dx = F(x) + C

其中 C 为任意常数，称为积分常数。这表示 f(x) 的所有原函数构成一个函数族。

1.2 基本积分公式

∫xndx = xn+1/(n+1) + C (n ≠ -1)
∫sin(x)dx = -cos(x) + C
∫cos(x)dx = sin(x) + C
∫exdx = ex + C
∫1/xdx = ln|x| + C
∫sec²(x)dx = tan(x) + C

1.3 不定积分的性质

∫[f(x) ± g(x)]dx = ∫f(x)dx ± ∫g(x)dx
∫k·f(x)dx = k∫f(x)dx (k为常数)

第二章：积分方法

2.1 换元积分法

第一类换元法（凑微分法）：

∫f[g(x)]g'(x)dx = ∫f(u)du (其中 u = g(x))

第二类换元法：

∫f(x)dx = ∫f[φ(t)]φ'(t)dt (其中 x = φ(t))

2.2 分部积分法

∫u·dv = uv - ∫v·du

分部积分法适用于被积函数是两个不同类型函数乘积的情况，如多项式与指数函数、三角函数等的乘积。

例题1：第一类换元法

题目：计算 ∫2x·cos(x²)dx

解答：
设 u = x²，则 du = 2xdx
∫2x·cos(x²)dx = ∫cos(u)du
= sin(u) + C
= sin(x²) + C

例题2：分部积分法

题目：计算 ∫x·exdx

解答：
设 u = x，dv = exdx
则 du = dx，v = ex
∫x·exdx = x·ex - ∫exdx
= x·ex - ex + C
= (x - 1)ex + C

第三章：定积分

3.1 定积分的定义

定积分是函数在区间上的累积和，几何上表示函数曲线与x轴之间的有向面积。

∫abf(x)dx = limn→∞ Σi=1nf(ξi)Δxi

其中 a 和 b 分别称为积分下限和上限。

微积分基本定理（牛顿-莱布尼茨公式）

如果 f(x) 在区间 [a, b] 上连续，F(x) 是 f(x) 的一个原函数，那么：

∫abf(x)dx = F(b) - F(a)

3.2 定积分的性质

∫abf(x)dx = -∫baf(x)dx
∫abf(x)dx = ∫acf(x)dx + ∫cbf(x)dx
∫abk·f(x)dx = k∫abf(x)dx

第四章：定积分的应用

4.1 平面图形的面积

由曲线 y = f(x)，x轴，直线 x = a，x = b 围成的面积：

A = ∫ab|f(x)|dx

由两条曲线 y = f(x) 和 y = g(x) 围成的面积：

A = ∫ab|f(x) - g(x)|dx

4.2 旋转体的体积

绕x轴旋转的体积：

V = π∫ab[f(x)]²dx

绕y轴旋转的体积：

V = 2π∫abx·f(x)dx

例题3：计算面积

题目：求由抛物线 y = x² 和直线 y = x + 2 围成的图形面积。

解答：
第一步：求交点
x² = x + 2
x² - x - 2 = 0
(x - 2)(x + 1) = 0
所以 x = -1 或 x = 2

第二步：确定上下函数
在区间 [-1, 2] 上，x + 2 > x²

第三步：计算面积
A = ∫-12[(x + 2) - x²]dx
= ∫-12(-x² + x + 2)dx
= [-x³/3 + x²/2 + 2x]-12
= [(-8/3 + 2 + 4) - (1/3 + 1/2 - 2)]
= [(-8/3 + 6) - (5/6 - 2)]
= [10/3 - (-7/6)] = 10/3 + 7/6 = 27/6 = 9/2

例题4：旋转体体积

题目：求曲线 y = √x，x = 0，x = 4 绕x轴旋转所成立体的体积。

解答：
V = π∫04(√x)²dx
= π∫04x dx
= π[x²/2]04
= π(16/2 - 0) = 8π

第五章：练习题

练习1：不定积分计算

计算下列不定积分：

  1. ∫(3x² - 2x + 1)dx
  2. ∫x·sin(x)dx
  3. ∫e2xdx
  4. ∫1/(x² + 1)dx
答案：
1. ∫(3x² - 2x + 1)dx = x³ - x² + x + C
2. ∫x·sin(x)dx = -x·cos(x) + sin(x) + C
3. ∫e2xdx = (1/2)e2x + C
4. ∫1/(x² + 1)dx = arctan(x) + C

练习2：定积分计算

计算下列定积分：

  1. ∫01(x² + 2x)dx
  2. ∫0π/2sin(x)dx
  3. ∫1e(1/x)dx
答案：
1. ∫01(x² + 2x)dx = [x³/3 + x²]01 = 1/3 + 1 = 4/3
2. ∫0π/2sin(x)dx = [-cos(x)]0π/2 = -0 + 1 = 1
3. ∫1e(1/x)dx = [ln|x|]1e = 1 - 0 = 1

练习3：应用题

求由曲线 y = x²，y = 0，x = 1 围成的图形绕x轴旋转所成立体的体积。

答案：
V = π∫01(x²)²dx
= π∫01x⁴dx
= π[x⁵/5]01
= π(1/5 - 0) = π/5
