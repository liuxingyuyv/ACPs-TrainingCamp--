第一章：极限的基本概念

1.1 极限的直观理解

极限描述的是当自变量无限接近某个值时，函数值的变化趋势。它关注的是"接近过程中的行为"，而不是在该点的实际值。

limx→af(x) = L

表示当 x 无限接近 a 时，f(x) 无限接近 L。

1.2 极限的严格定义（ε-δ定义）

对于任意正数 ε > 0，存在正数 δ > 0，使得当 0 < |x - a| < δ 时，有 |f(x) - L| < ε，则称 limx→af(x) = L。

∀ε > 0, ∃δ > 0, s.t. 0 < |x - a| < δ ⇒ |f(x) - L| < ε

1.3 极限的类型

  • 双侧极限：limx→af(x) = L
  • 左极限：limx→a⁻f(x) = L
  • 右极限：limx→a⁺f(x) = L
  • 无穷极限：limx→∞f(x) = L 或 limx→af(x) = ∞

重要定理：limx→af(x) = L 当且仅当 limx→a⁻f(x) = limx→a⁺f(x) = L

第二章：极限的性质与运算法则

2.1 极限的基本性质

  • 唯一性：如果极限存在，则极限值唯一
  • 局部有界性：如果 limx→af(x) = L，则 f(x) 在 a 的某邻域内有界
  • 保号性：如果 limx→af(x) = L > 0，则在 a 的某邻域内 f(x) > 0

2.2 极限的运算法则

如果 limx→af(x) = A，limx→ag(x) = B，则：

limx→a[f(x) ± g(x)] = A ± B
limx→a[f(x) · g(x)] = A · B
limx→a[f(x) / g(x)] = A / B (B ≠ 0)
limx→a[f(x)]n = An

2.3 夹逼定理

如果在 a 的某邻域内，g(x) ≤ f(x) ≤ h(x)，且 limx→ag(x) = limx→ah(x) = L，则 limx→af(x) = L。

第三章：重要极限

3.1 两个重要极限

重要极限一：

limx→0sin(x)/x = 1

重要极限二：

limx→∞(1 + 1/x)x = e
limx→0(1 + x)1/x = e

3.2 常见极限公式

limx→0tan(x)/x = 1
limx→0(1 - cos(x))/x² = 1/2
limx→0(ex - 1)/x = 1
limx→0ln(1 + x)/x = 1

第四章：极限的计算方法

4.1 直接代入法

当函数在极限点连续时，可以直接代入求值。

4.2 因式分解法

对于 0/0 型不定式，通过因式分解约去公因子。

4.3 有理化法

对于含有根号的不定式，通过有理化消除根号。

4.4 洛必达法则

对于 0/0 或 ∞/∞ 型不定式，可以使用洛必达法则：

limx→af(x)/g(x) = limx→af'(x)/g'(x)

（当右端极限存在时）

例题1：因式分解法

题目：计算 limx→2(x² - 4)/(x - 2)

解答：
直接代入得到 0/0 型不定式
分子因式分解：x² - 4 = (x - 2)(x + 2)
limx→2(x - 2)(x + 2)/(x - 2) = limx→2(x + 2) = 4

例题2：有理化法

题目：计算 limx→0(√(x + 1) - 1)/x

解答：
分子有理化：乘以(√(x + 1) + 1)/(√(x + 1) + 1)
= limx→0[(x + 1) - 1]/[x(√(x + 1) + 1)]
= limx→0x/[x(√(x + 1) + 1)]
= limx→01/(√(x + 1) + 1) = 1/2

例题3：重要极限应用

题目：计算 limx→0sin(3x)/x

解答：
利用重要极限 limx→0sin(x)/x = 1
limx→0sin(3x)/x = limx→0[sin(3x)/(3x)]·3
= 1·3 = 3

例题4：洛必达法则

题目：计算 limx→0(ex - 1)/x

解答：
直接代入得到 0/0 型不定式
应用洛必达法则：
limx→0(ex - 1)/x = limx→0ex/1 = e0 = 1

第五章：无穷小与无穷大

5.1 无穷小的概念

如果 limx→af(x) = 0，则称 f(x) 为当 x→a 时的无穷小量。

无穷小的比较：

  • 如果 limx→aβ/α = 0，则称 β 是比 α 高阶的无穷小
  • 如果 limx→aβ/α = c ≠ 0，则称 β 与 α 是同阶无穷小
  • 如果 limx→aβ/α = 1，则称 β 与 α 是等价无穷小

5.2 常见等价无穷小

当 x→0 时：

sin(x) ~ x
tan(x) ~ x
1 - cos(x) ~ x²/2
ex - 1 ~ x
ln(1 + x) ~ x

第六章：连续性

6.1 连续性的定义

函数 f(x) 在点 a 处连续，当且仅当满足以下三个条件：

  1. f(a) 存在
  2. limx→af(x) 存在
  3. limx→af(x) = f(a)

6.2 间断点的类型

  • 可去间断点：左右极限存在且相等，但不等于函数值
  • 跳跃间断点：左右极限存在但不相等
  • 无穷间断点：至少有一侧极限为无穷大
  • 振荡间断点：极限不存在且不为无穷大

第七章：练习题

练习1：基本极限计算

计算下列极限：

  1. limx→3(x² - 9)/(x - 3)
  2. limx→0sin(2x)/x
  3. limx→∞(2x² + 3x + 1)/(x² - x + 2)
  4. limx→0(ex - 1 - x)/x²
答案：
1. limx→3(x² - 9)/(x - 3) = limx→3(x + 3) = 6
2. limx→0sin(2x)/x = limx→0[sin(2x)/(2x)]·2 = 2
3. limx→∞(2x² + 3x + 1)/(x² - x + 2) = 2
4. limx→0(ex - 1 - x)/x² = limx→0ex/2 = 1/2 (洛必达法则)

练习2：有理化计算

计算 limx→4(√x - 2)/(x - 4)

答案：
分子有理化：
limx→4(√x - 2)(√x + 2)/(x - 4)(√x + 2)
= limx→4(x - 4)/(x - 4)(√x + 2)
= limx→41/(√x + 2) = 1/4

练习3：连续性判断

判断函数 f(x) = { x², x ≤ 1 2x - 1, x > 1 } 在 x = 1 处是否连续。

答案：
检查连续性条件：
1. f(1) = 1² = 1
2. limx→1⁻f(x) = limx→1⁻x² = 1
3. limx→1⁺f(x) = limx→1⁺(2x - 1) = 1
因为左右极限相等且等于函数值，所以函数在 x = 1 处连续。
