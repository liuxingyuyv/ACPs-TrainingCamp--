第一章：物理学中的应用

1.1 运动学问题

微积分在物理学中最直接的应用是描述物体的运动。

速度：v(t) = ds/dt
加速度：a(t) = dv/dt = d²s/dt²
位移：s(t) = ∫v(t)dt

案例1：自由落体运动

问题：一个物体从高度 h 处自由落下，忽略空气阻力，求：

  1. 物体在任意时刻 t 的速度
  2. 物体到达地面所需的时间
  3. 物体到达地面时的速度
解答：
已知重力加速度 g = 9.8 m/s²

1. 速度：v(t) = ∫a dt = ∫g dt = gt + C₁
初始条件：v(0) = 0，所以 C₁ = 0
v(t) = gt

2. 位移：s(t) = ∫v dt = ∫gt dt = (1/2)gt² + C₂
初始条件：s(0) = 0，所以 C₂ = 0
s(t) = (1/2)gt²
到达地面时：s(t) = h
(1/2)gt² = h
t = √(2h/g)

3. 到达地面时的速度：
v = gt = g√(2h/g) = √(2gh)

1.2 功与能量

变力做功的计算需要用到积分：

W = ∫abF(x)dx

其中 F(x) 是随位置变化的力。

案例2：弹簧的弹性势能

问题：根据胡克定律，弹簧的弹力 F = -kx，其中 k 是劲度系数，x 是伸长量。求将弹簧从原长拉伸到长度 L 时所做的功。

解答：
克服弹力做功：W = ∫0Lkx dx
= [kx²/2]0L
= kL²/2
这就是弹簧的弹性势能公式。

第二章：经济学中的应用

2.1 边际分析

在经济学中，导数用来描述边际量：

  • 边际成本：MC = dC/dQ（成本对产量的导数）
  • 边际收益：MR = dR/dQ（收益对产量的导数）
  • 边际利润：MP = dP/dQ（利润对产量的导数）

案例3：最优产量决策

问题：某公司的成本函数为 C(Q) = 1000 + 10Q + 0.01Q²，收益函数为 R(Q) = 50Q - 0.02Q²，求利润最大时的产量。

解答：
利润函数：P(Q) = R(Q) - C(Q)
= (50Q - 0.02Q²) - (1000 + 10Q + 0.01Q²)
= -1000 + 40Q - 0.03Q²

边际利润：MP = dP/dQ = 40 - 0.06Q
令 MP = 0：40 - 0.06Q = 0
Q = 40/0.06 ≈ 666.67

验证二阶导数：d²P/dQ² = -0.06 < 0
所以 Q ≈ 667 时利润最大。

2.2 消费者剩余与生产者剩余

积分用于计算市场均衡时的剩余：

消费者剩余 = ∫0Q*[D(Q) - P*]dQ
生产者剩余 = ∫0Q*[P* - S(Q)]dQ

第三章：生物学中的应用

3.1 种群增长模型

微分方程描述种群动态：

指数增长：dN/dt = rN
逻辑斯谛增长：dN/dt = rN(1 - N/K)

其中 N 是种群数量，r 是增长率，K 是环境容量。

案例4：细菌繁殖

问题：某种细菌按指数规律增长，增长率 r = 0.5/小时。如果初始数量为 1000，求 6 小时后的细菌数量。

解答：
解微分方程：dN/dt = 0.5N
dN/N = 0.5dt
∫dN/N = ∫0.5dt
ln|N| = 0.5t + C
N = Ce0.5t
初始条件：N(0) = 1000，所以 C = 1000
N(t) = 1000e0.5t
N(6) = 1000e3 ≈ 1000 × 20.086 = 20086

第四章：工程学中的应用

4.1 优化问题

工程中的许多设计问题都可以转化为优化问题：

  • 材料最省设计
  • 成本最低方案
  • 效率最大配置

案例5：最优容器设计

问题：设计一个体积为 V = 1000 cm³ 的圆柱形罐头，如何选择底面半径 r 和高 h 使材料最省（表面积最小）？

解答：
约束条件：V = πr²h = 1000
所以 h = 1000/(πr²)

表面积：S = 2πr² + 2πrh
= 2πr² + 2πr × 1000/(πr²)
= 2πr² + 2000/r

求导：dS/dr = 4πr - 2000/r²
令 dS/dr = 0：4πr = 2000/r²
4πr³ = 2000
r³ = 500/π
r ≈ 5.42 cm

h = 1000/(π × 5.42²) ≈ 10.84 cm

验证：d²S/dr² = 4π + 4000/r³ > 0
所以这个设计使表面积最小。

第五章：统计学中的应用

5.1 概率密度函数

连续型随机变量的概率分布用密度函数描述：

P(a ≤ X ≤ b) = ∫abf(x)dx
期望：E(X) = ∫-∞∞xf(x)dx
方差：Var(X) = ∫-∞∞(x - μ)²f(x)dx

案例6：正态分布概率计算

问题：设 X ~ N(μ, σ²)，求 P(μ - σ ≤ X ≤ μ + σ)。

解答：
标准化：Z = (X - μ)/σ ~ N(0, 1)
P(μ - σ ≤ X ≤ μ + σ) = P(-1 ≤ Z ≤ 1)
= ∫-11(1/√(2π))e-z²/2dz
≈ 0.6826

这就是著名的"68-95-99.7法则"中的68%。

第六章：跨学科综合案例

案例7：传染病传播模型（SIR模型）

问题：建立传染病传播的数学模型，预测疫情发展趋势。

解答：
SIR模型将人群分为三类：
• S(t)：易感者数量
• I(t)：感染者数量
• R(t)：康复者数量

微分方程组：
dS/dt = -βSI/N
dI/dt = βSI/N - γI
dR/dt = γI

其中 β 是感染率，γ 是康复率，N 是总人口。

这个模型可以通过数值方法求解，预测疫情峰值和持续时间。

案例8：金融期权定价（Black-Scholes模型）

问题：计算欧式看涨期权的理论价格。

解答：
Black-Scholes公式：
C = S₀N(d₁) - Ke-rTN(d₂)

其中：
d₁ = [ln(S₀/K) + (r + σ²/2)T]/(σ√T)
d₂ = d₁ - σ√T

N(d) 是标准正态分布的累积分布函数：
N(d) = (1/√(2π))∫-∞de-t²/2dt

这个公式涉及导数（对冲率）、积分（概率计算）和极限（连续时间假设）。

第七章：练习题

练习1：物理应用

一个质点沿直线运动，其速度函数为 v(t) = 3t² - 2t + 1 (m/s)。求：

  1. t = 2s 时的加速度
  2. 前3秒内质点的位移
  3. 质点何时速度为零
答案：
1. 加速度：a(t) = dv/dt = 6t - 2
a(2) = 6×2 - 2 = 10 m/s²

2. 位移：s(t) = ∫v dt = ∫(3t² - 2t + 1)dt = t³ - t² + t + C
假设 s(0) = 0，则 C = 0
s(3) = 27 - 9 + 3 = 21 m

3. 令 v(t) = 0：3t² - 2t + 1 = 0
判别式 Δ = 4 - 12 = -8 < 0
所以速度恒不为零。

练习2：经济应用

某产品的需求函数为 P = 100 - 2Q，供应函数为 P = 20 + Q。求市场均衡时的消费者剩余。

答案：
均衡时：100 - 2Q = 20 + Q
3Q = 80，Q = 80/3 ≈ 26.67
P = 100 - 2×(80/3) = 100 - 160/3 = 140/3 ≈ 46.67

消费者剩余 = ∫080/3(100 - 2Q - 140/3)dQ
= ∫080/3(160/3 - 2Q)dQ
= [160Q/3 - Q²]080/3
= 160/3 × 80/3 - (80/3)²
= 12800/9 - 6400/9 = 6400/9 ≈ 711.11

练习3：优化问题

要建造一个容积为 500 m³ 的无盖长方体水池，如何设计尺寸使材料最省？

答案：
设底面长为 x，宽为 y，高为 h
约束：xyh = 500
表面积：S = xy + 2xh + 2yh

由约束：h = 500/(xy)
S = xy + 2x×500/(xy) + 2y×500/(xy)
= xy + 1000/y + 1000/x

求偏导并令其为零：
∂S/∂x = y - 1000/x² = 0
∂S/∂y = x - 1000/y² = 0

解得：x = y = 10 m
h = 500/(10×10) = 5 m

最优设计：底面 10m × 10m，高 5m。
